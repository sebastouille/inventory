# Vue d'ensemble du projet

## Objet

Inventory App est une application d'inventaire patrimonial connectee a Archicad et SINERGI/SAP. Elle couvre l'inventaire physique, la localisation, les campagnes, les anomalies, les mouvements, le rapprochement, l'audit, et la securite nominative.

## Etat actuel du produit

- Monorepo avec `apps/api`, `apps/web`, `apps/admin`, `packages/shared`, et `prisma/`
- API actuellement implementee en NestJS avec Prisma et PostgreSQL
- Frontend operateur en Next.js
- Back-office admin en Next.js
- Workflow documentaire versionne dans le repo
