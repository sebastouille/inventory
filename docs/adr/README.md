# Convention ADR

Utiliser les Architecture Decision Records pour les choix qui modifient l'architecture, les frontieres runtime, la securite, les integrations, la gouvernance API, ou la strategie de donnees.

## Nommage

- format : `NNNN-titre-kebab-case.md`
- exemple : `0001-documentation-governance-workflow.md`

## Sections obligatoires

- statut
- contexte
- decision
- consequences
- alternatives considerees
- actions de suivi
