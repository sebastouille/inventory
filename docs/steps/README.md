# Dossiers d'etape

Creer un fichier par fonctionnalite majeure, migration, ou etape de conception.

## Nommage

- format : `YYYY-MM-DD-titre-kebab-case.md`
- exemple : `2026-06-12-documentation-governance-bootstrap.md`

## Workflow obligatoire

Avant implementation :

- creer le fichier d'etape ;
- capturer l'objectif, le perimetre, le plan, les questions, les hypotheses, les zones impactees, les tests, et les criteres d'acceptation.

Pendant implementation :

- deplacer les points des questions ouvertes vers les decisions prises ;
- mettre a jour les impacts quand ils deviennent concrets.

Apres implementation :

- enregistrer ce qui a ete livre ;
- enregistrer les tests executes ;
- enregistrer les ecarts restants et le suivi backlog.
