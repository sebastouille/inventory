# Carte du workflow

## Cas de declenchement

Utiliser la skill quand une demande :

- lance une nouvelle fonctionnalite ou une nouvelle etape ;
- demande un plan avant implementation ;
- modifie l'architecture, la securite, le schema, ou le deploiement ;
- cree des questions ouvertes ou du travail differe ;
- demande des decisions tracables ou des tickets backlog.

## Artefacts attendus

- un dossier d'etape dans `docs/steps/`
- zero ou plusieurs ADR dans `docs/adr/`
- des mises a jour des documents racine
- des entrees backlog si le scope est differe

## Sequencement recommande

1. lire la documentation courante
2. creer ou mettre a jour le fichier d'etape
3. implementer ou analyser
4. mettre a jour les documents de reference
5. enregistrer ADR ou backlog
